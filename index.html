Some things I've worked on

BatHunt

BatHunt is inspired by the NES game DuckHunt. It is the first game I have completed, and it's playable at https://slimymelon.itch.io/bathunt

Pairless

Pairless is a game that I've worked on for a school project with some other students. It's a top-down RPG game that doesn't take itself too serious. The game can be found at https://sleepy-wolf-studios.itch.io/pairless

PatternPal

PatternPal is a Visual Studio extension that can recognize software design patterns in your code. It also contains a step-by-step mode for learning new design patterns. I have worked on this project in 2021 with other students from Hogeschool Windesheim. https://github.com/PatternPal/PatternPal
